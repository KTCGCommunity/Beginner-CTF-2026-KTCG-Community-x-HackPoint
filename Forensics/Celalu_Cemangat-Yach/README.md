## Description
Sebuah file ZIP terenkripsi ditemukan di server lama milik divisi keamanan.  

Beberapa petunjuk yang berhasil ditemukan:

- Password diawali dengan `BT`
- Panjang password hanya 4 karakter
- 2 karakter terakhir berupa angka
- Tools yang mungkin berguna: the ripper dan sesuatu yang menghasilkan suara gigitan renyah

Bisakah kamu membobol file tersebut dan mendapatkan flag?

## Flag Format
`KTCG{...}`

---

## Flag
`KTCG{r1pp3d_by_j0hn}`
