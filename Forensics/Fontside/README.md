## Category
Forensics

## Difficulty

## Author

## Description
Sebuah workstation milik desainer berhasil diamankan setelah insiden kebocoran data internal.  

Di dalamnya ditemukan beberapa file font OpenType (`.otf`) yang tampak normal dan masih dapat digunakan seperti biasa.

Namun, tim forensik mencurigai bahwa file-file tersebut telah dimodifikasi untuk menyembunyikan suatu pesan rahasia.  

Mungkin jawabannya tidak berada pada tampilan fontnya...  
melainkan pada struktur internalnya.
