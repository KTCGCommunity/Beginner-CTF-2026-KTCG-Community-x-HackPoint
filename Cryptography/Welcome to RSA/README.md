## Category
Cryptography

Easy

## Author

## Description
Sebuah sistem enkripsi RSA digunakan untuk mengamankan sebuah pesan rahasia. Namun karena kesalahan konfigurasi, dua bilangan prima penting dalam proses pembentukan kunci berhasil ter-leak.  

Tugasmu adalah memanfaatkan informasi tersebut untuk merekonstruksi parameter RSA, menghitung nilai yang dibutuhkan, dan mendekripsi ciphertext yang diberikan untuk mendapatkan flag. 🔐  

Challenge ini menguji pemahaman dasar mengenai:

- Perhitungan modulus RSA (`N = p × q`)
- Totient Euler
- Perhitungan private key
- Proses dekripsi RSA

Good luck and happy cracking!
