# Window After Rain

## Category
OSINT

## Difficulty
Medium

## Author
KTCG Community

## Description
Sebuah foto diambil dari lantai atas sebuah hotel saat hujan baru saja reda.

Beberapa petunjuk tersembunyi di sekitar jalanan, bangunan, hingga arah penunjuk jalan yang terlihat samar.

Tugasmu sederhana:  
identifikasi hotel tempat foto ini diambil, lalu temukan plus code hotel tersebut dan lantai berapa foto ini diambil.

## Flag Format
`KTCG{pluscode_lantai}`

Contoh:
`KTCG{33W9+35_100}`
