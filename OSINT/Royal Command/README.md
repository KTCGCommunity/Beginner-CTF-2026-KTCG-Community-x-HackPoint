## Description
Sebuah kapal kerajaan legendaris pernah digunakan dalam berbagai perjalanan penting dan acara kenegaraan.

Di balik kemewahan tersebut, terdapat seorang perwira tinggi Angkatan Laut Kerajaan yang memimpin kapal itu pada masanya.

Dari arsip publik yang tersedia di internet, cari tahu siapa yang menjabat sebagai Flag Officer Royal Yachts (FORY) pada periode 1962–1965.
